
🖼️ Mengganti Foto:
Letakkan foto Niswatun di folder yang sama, beri nama niswatun.jpg, lalu buka file HTML dengan teks editor dan ganti baris ini:
src="https://via.placeholder.com/..."
menjadi:
src="niswatun.jpg"

🎵 Menambahkan Musik:
Letakkan file music.mp3 di folder yang sama, lalu di dalam file HTML hapus tanda komentar pada baris:
html<!-- <source src="music.mp3" type="audio/mpeg"> -->
menjadi:
html<source src="music.mp3" type="audio/mpeg">

Cara untuk deploy gratis:

Opsi 1: Netlify Drop (paling mudah, 1 menit)
	Buka netlify.com/drop
	Drag & drop file ulang-tahun-niswatun.html ke sana
	Dapat link seperti: https://amazing-name-123.netlify.app
	Kirim link itu ke Niswatun 🎉

Opsi 2: GitHub Pages (gratis, permanen)
	Buat akun GitHub → buat repo baru
	Upload file HTML, rename jadi index.html
	Aktifkan GitHub Pages di Settings
	Dapat link: https://usernamekamu.github.io/nama-repo

Opsi 3: Tiiny.host (paling simpel)
	Buka tiiny.host
	Upload file HTML
	Langsung dapat link publik
